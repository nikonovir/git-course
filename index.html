GIT flow

GIT clone - для того, щоб "скопіювати" репозиторій
GIT pull - для того, щоб отримати зміни с репозиторія
GIT checkout-b branchname - создать новою ветку і переключитися на
GIT add . - добавить зміни, які зробили локально
GIT commit-m "commit text" - підписати їх
GIT push origin branchname - запушить на git сервер
